A SECURE ACTIONSCRIPT INTERPRETER IN COQ:

Step 1 (Installation):
Installing Coq IDE is a primary and mandatory step for executing the source code which is written in Coq Theorem Proving System.

Step 2 (Execution):
In this project there are 8 files. Execute the file "AS_Init.v" which loads all the dependent files by itself.
In this file, there are 3 different instruction sequences demonstrating the overflow conditions viz 
instruction_sequence, instruction_sequence1, instruction_sequence2.
